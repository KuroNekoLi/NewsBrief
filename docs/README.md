## 🚀 快速開始

1. **克隆專案**

```bash
git clone <repository-url>
cd NewsBrief
```

2. **安裝依賴**

```bash
npm install
cd ios && pod install && cd ..
```

3. **運行專案**

```bash
# iOS
npm run ios

# Android
npm run android
```

4. **開始學習**
   選擇適合你的學習路徑，從對應的教學文件開始。
